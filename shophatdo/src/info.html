Cách tính phí

GIÁ VỀ VN = (Giá web + Công mua + Ship Web) x tỷ giá + Ship Nhật Việt + Ship VN

Giá web : Giá sản phẩm trên web

Công mua : 8% giá web ( khách mua buôn liên hệ với shop để được giá ưu đãi )

Ship Web : tiền ship nội địa của Nhật ( nếu có )

Tỷ giá : 245 (1 yên đổi 245 VND)

Ship Nhật Việt : tiền ship từ Nhật về Việt Nam (xem thêm mục 2.1)

Ship VN : tiền ship nội địa tại Việt Nam (xem thêm mục 2.2)

Cách tính tiền ship Nhật Việt :

Ngoại trừ các mặt hàng cồng kềnh, hàng trị giá cao, hàng vụn giá trị nhỏ, hàng điện tử với mức phí ship riêng tùy từng đợt chuyển hàng, phí ship Nhật Việt đối với các mặt hàng thông thường khác là 250k/ 1kg.

Phí Ship VN:

  • Nội thành Hà Nội: 17k-25K cho 1 đơn hàng (1 địa điểm vận chuyển); miễn phí cho đơn hàng trên 2 triệu.

  • Ngoại thành Hà Nội: áp dụng theo bảng giá EMS - dịch vụ chuyển phát nhanh giá rẻ của bưu điện (liên hệ với shop để biết thêm chi tiết).
